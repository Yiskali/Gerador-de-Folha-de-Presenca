# Gerador de Folha de Presença

# O que é
Ferramenta interna criada para facilitar a emissão de folhas de presença para assinatura durante períodos de substituição em cargos de chefia na Ebserh, sem necessidade de usar Excel.

# Objetivo
Simplificar o preenchimento, padronizar as folhas de presença e reduzir erros manuais e retrabalho.

# Público
Uso interno, destinado a empregados e setores autorizados da EBSERH.

# Funcionamento
A ferramenta recebe os dados necessários e gera automaticamente a folha de presença, permitindo a exportação do arquivo PDF para uso interno.

# Status
Em fase de testes.

# Acesso
Este repositório é privado. O código não deve ser copiado, compartilhado ou divulgado externamente sem autorização.

# Observação
